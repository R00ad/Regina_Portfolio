Project 1: IoT Security Alarm System

Project scope

IoT Security Alarm System

An alarm system that can automatically detect suspicious activity in real time while still allowing authorised users to enter and exit without triggering alarms using IoT sensors, Arduino IDE and Linux OS

Solution

My thought process

Our solution is a webpage that monitors real values that changes as time passes, and code HTML, flask, and upload our data detected from sensors into the MariaDB and then display it on the webpage.

Work process

Sensors: Light and Sound sensor to monitor changes.

Raspberry Pi: Compare readings from sensor to threshold and determine what state the room is.

Database: Stores processed readings, timestamps, and status labels in MariaDB. Keeps both real‑time and historical records for tracking events in case of intrusion.

Web Server Flask server retrieves and processes database data for display.

Project 2: Wheelchair Grabber

Project scope

Wheelchair Grabber

To design a prototype which can help people on wheelchairs:​ to reach for items on places they can't reach at ease, ​ without having to ask others for assistance.

Solution

My thought process

We chose to design a grabber to assist individuals with mobility challenges in reaching items on high shelves. We chose this as our finalised idea as:​ It’s the most practical and realistic idea​ It brings high value and at the same time requiring low effort (value-effort map)​

My Work process

Arduino and tinkercad: to code the grabbing motion which are controlled by a button

Circuit connection: to be enable the motor to open and close the jaw + button activation

2 prototype: improving the grabber to maximise ease of usage of the grabber for users.
